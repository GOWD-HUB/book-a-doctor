# Book a Doctor

A full-stack MERN application for booking doctor appointments.

## Features

- User Authentication
- Admin Dashboard
- Doctor Dashboard
- Patient Dashboard
- Book Appointment
- View Appointments
- Profile Management
- Forgot Password

## Tech Stack

Frontend:
- React
- React Router
- Axios
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
